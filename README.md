## Project Setup

Run the following commands in project root directory.

**Linux**
```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements. txt 
```

**Windows**
```
python3 -m venv .venv
venv/Scripts/activate
pip install -r requirements. txt 
```
